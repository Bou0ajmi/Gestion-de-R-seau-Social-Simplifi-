# Gestion_de_Réseau_Social_Simplifié.

## une application console pour simuler la gestion des relations (amis,abonnements) et le flux de publications.

*L’application crée son propre réseau et gère les données des utilisateurs.

*L’utilisateur peut manipuler son compte : ajouter des relations, publier des publications et consulter celles des autres.

*L’interaction avec l’utilisateur se fait dans le terminal : un menu d’actions s’affiche à l’ouverture de l’application, et l’utilisateur est libre de choisir ce qu’il veut faire dans le réseau.

 
*La sauvegarde des données se fait automatiquement lorsque l’utilisateur ferme l’application, dans deux fichiers : « relations.txt » pour les relations, et « utilisateurs.bin » pour les données spécifiques de chaque utilisateur.

*Le code constitue un bon exercice pour se familiariser avec les différentes structures de données et apprendre à les combiner entre elles.(arbre binaire, pile, file...)

### installation:

1.cloner le poject

2.l'ouvrir dans n'importe quel Compilateur C
